# 다국어 지원 (i18n)

## 라우터 설정

```javascript
const router = new ViewLogicRouter({
    useI18n: true,
    defaultLanguage: 'ko'
});
```

## 메시지 파일 구조

```
project/
└── i18n/
    ├── ko.json
    ├── en.json
    └── ja.json
```

### 메시지 파일 예시 (ko.json)
```json
{
    "welcome": "환영합니다",
    "hello": "안녕하세요, {name}님",
    "product": {
        "title": "제품",
        "description": "설명"
    }
}
```

## HTML에서 사용

```html
<h1>{{ $t('welcome') }}</h1>
<p>{{ $t('hello', { name: userName }) }}</p>
<p>{{ $t('product.title') }}</p>
```

## JavaScript에서 사용

```javascript
const msg = this.$t('welcome');
const currentLang = this.$lang;
```

## 언어 전환

```javascript
this.$i18n.setLanguage('en');
```

```html
<button @click="$i18n.setLanguage('ko')" :class="{ active: $lang === 'ko' }">한국어</button>
<button @click="$i18n.setLanguage('en')" :class="{ active: $lang === 'en' }">English</button>
```
